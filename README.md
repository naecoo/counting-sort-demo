# 计数排序算法可视化

这个项目使用manim库（3Blue1Brown的数学动画引擎）创建了一个计数排序算法的可视化讲解。

## 项目说明

计数排序是一种非比较排序算法，其时间复杂度为O(n+k)，其中k是数值范围。这个动画详细展示了计数排序的工作原理，包括：

1. 计数数组的创建和填充
2. 累积计数的计算
3. 输出数组的构建过程
